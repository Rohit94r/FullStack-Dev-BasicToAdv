## Project (6 hours): Personal Portfolio Page
Build in a new `project/` folder here. NO frameworks, NO Tailwind — pure HTML + CSS.

Required sections:
- [ ] Sticky navbar (flexbox, logo left + links right)
- [ ] Hero section (name, title, CTA button — centered with flexbox)
- [ ] About section (media-object pattern: image + text)
- [ ] Skills grid (CSS Grid, auto-fill responsive — no media query!)
- [ ] Projects section (3 cards with hover effects + transitions)
- [ ] Contact form (all input types, labels, validation attributes)
- [ ] Footer (semantic, copyright with `<time>`)
- [ ] Fully responsive: looks good at 375px, 768px, 1280px
- [ ] Dark theme using CSS variables (bonus: `prefers-color-scheme`)
- [ ] At least one keyframe animation (fade-in hero)

**Project scaffold ready:** `project/portfolio/` — fill in every `TODO` before checking answers.
