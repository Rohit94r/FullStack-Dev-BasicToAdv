## How to use this curriculum

1. Open `README.md` in the root — see the full 18-day map
2. Start with `day01-html-css/` — work in order
3. Each day has:
   - `README.md` — today's plan + checklist
   - `lessons/` — read, run, modify every file
   - `project/` — fill-in-the-blank (TODO → YOUR IDEA → ANSWER)
   - `interview-questions.md` — answer out loud every night
   - `notes.md` — write YOUR revision notes every night

## Daily workflow (15 hours)

| Block | Hours | Do this |
|-------|-------|---------|
| Learn | 4 | Run every lesson file, change values, break things |
| Revise | 2 | Re-do yesterday's hardest topic from memory |
| Build | 6 | Complete the day's project TODOs |
| Docs | 2 | Official docs for today's topics |
| Notes | 1 | `notes.md` + `interview-questions.md` |
